*************
Installation
*************
You can clone the `repository <https://gitlab.kuleuven.be/meco-software/impact>`_  by calling

    .. code:: bash

        git clone git@gitlab.kuleuven.be:meco-software/impact.git

To install the toolchain just run the setup.py script with the install parameter. It will install Tasho and also download and install all dependencies:

    .. code:: bash

       python setup.py install

.. note::

   You can also install IMPACT via `pip install impact-meco`
